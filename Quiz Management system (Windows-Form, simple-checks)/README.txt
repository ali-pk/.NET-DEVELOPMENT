About this mini-project:
	1) This is .net windows form project, in which we maintained sessions.
	2) For every correct answer, score is incremented.
	3) If the selected answer is correct, then move to next question.
	4) There is also a timer of 1 minute for a whole session.
	5) User can answer atleast 5 questions.
	6) The questions and their options are fetched from database and the choosen answer 	   is checked from the answer in database.
	7) For more details and secreenshots check the PROJECT-REPORT.pdf in folder
